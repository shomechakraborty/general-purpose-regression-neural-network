# general-purpose-regression-neural-network

Programmed By: Shome Chakraborty

INTRODUCTION

- This program is a general-purpose regression neural network model that can be used to predict target values using data in a variety of quantitative applications. This model uses Stochastic Gradient Descent (SGD) and is designed to capture and learn complex relationships and patterns in quantitative data, thereby developing robust predictive ability beyond traditional regression. Users can provide the model training data and several hyperparameters for the specific predictive application that the model will train on. Users can run their custom-trained model on input features to obtain predictive values for their use case. The model is designed to be fine-tuned to a user’s application and preferences while enabling optimized regression training and data analysis. 
- This model is being used towards public health capacity forecasting research available at this repository: https://github.com/shomechakraborty/public_health_forecasting_using_computationally_efficient_models.git
